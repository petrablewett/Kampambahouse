IMAGE GUIDELINES:
----------------
- Format: JPG (JPEG) preferred
- Quality: High resolution, well-lit, professional
- Style: Natural, warm, inviting
- Orientation: Most should be landscape (horizontal)
- Size: Optimize for web (100-500 KB per image after compression)
- Naming: Use exact filenames listed above

TOTAL IMAGES NEEDED: 27+

Optional: Add more gallery images to showcase your property!


TIPS FOR GREAT PHOTOS:
---------------------
✓ Shoot during golden hour (sunrise/sunset) for warm lighting
✓ Keep photos bright and airy
✓ Show the cottage interiors clean and well-staged
✓ Include shots of the views guests will enjoy
✓ Capture the peaceful, relaxing atmosphere
✓ Show unique features and amenities
✓ Include some detail shots (wine glasses, decor, etc.)
✓ Make sure photos are sharp and in focus


IMAGE OPTIMIZATION:
------------------
Before uploading, compress images to reduce file size:
- Use tools like TinyPNG.com or Squoosh.app
- Target 100-300 KB per image
- Maintain visual quality while reducing file size
- This helps your website load faster
